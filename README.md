# CS 418 Project 2: Regression, Classification, and Clustering

## Project Description  
This project applies regression, classification, and clustering techniques to analyze how demographic factors such as income, education, population, age, and race relate to voting patterns across U.S. counties. It also evaluates how well these methods can predict election outcomes and group counties with similar political behavior.
